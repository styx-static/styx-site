Styx 0.3.0 released

2016-10-26

Highlights:

  • Themes as first class citizens
  • New themes
  • Styx library not required in site.nix
  • Taxonomies
  • New functions in library, new sub libraries
  • AsciiDoc support
  • Multipages
  • Easier updates

Learn more in the Styx 0.3.0 Documentation, release notes.
